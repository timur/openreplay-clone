What is a cloud AI agent?

A cloud AI agent is an agent that runs in the cloud instead of on your own device. In Bloome (Beta), your agent runs in an E2B cloud sandbox, so it stays online when your laptop is closed and is reachable from any device. It can run code and read or write files inside that sandbox.

Prefer running an agent on your own machine? See the personal AI agent

Cloud agent vs local agent

Both are the same Bloome agent — the difference is where it runs. Cloud hosting is Beta.

Cloud agent (Beta)

  • Runs in an E2B cloud sandbox
  • Stays online when your client app is closed
  • Reachable from any device — web, desktop, or phone
  • Runs code and read-writes files in its sandbox

Local agent

  • Runs on your own machine, tied to the client
  • Only active while your client app is open
  • Bound to the device the client runs on
  • Stops working the moment you close the app

FAQ

1.Is the cloud agent generally available?

No. Cloud hosting for your agent is in Beta — it appears as "In the cloud (Beta)" in Bloome. The local agent is the default. Behavior and availability can change while it is in Beta.

2.What does "always on" mean for a cloud AI agent?

It means the agent runs in the cloud and stays online when your client app or laptop is closed, so you can reach it from any device. Like any service, uptime is not guaranteed to be perfect, especially during Beta.

3.Where does the cloud agent run?

In an E2B cloud sandbox managed by Bloome. The agent lives there instead of on your laptop or phone, which is what lets it stay online and be reachable from any device.

4.Can the cloud agent run code and edit files?

Yes. Inside its sandbox the agent can execute shell commands and read, write, and edit files. That work happens in the cloud sandbox rather than on your own machine.

5.How is a cloud agent different from a local agent?

A local agent runs on your machine and only works while your client app is open. A cloud agent runs in an E2B sandbox, stays online when the app is closed, and is reachable from any device.

6.Is Bloome free to use?

Bloome is free to start — sign up and you get a personal agent immediately. Usage is credit-based, and you top up credits as you need them.
