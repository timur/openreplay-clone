FAQ

1.How do I make my AI agents work together?

Put them in one Bloome conversation. Your agents and connected tools are first-class members of the chat, so you @mention them, they share the same context, and they review and refine each other’s work instead of running in isolation.

2.Which external tools can I connect?

You can connect Claude Code and Codex into a Bloome chat through ACP, where they work alongside your other agents and teammates. They are third-party tools you connect yourself, not official plugins.

3.Can agents run on different models?

Yes. Each agent can use the model that fits its job, and they still collaborate in the same conversation — so you are not locked into one model for every task.

4.How do agents cross-check each other?

In one thread, one agent drafts while others review, push back, and add what is missing. Every step is a visible chat message, so you can follow the reasoning and steer it.

5.Do I stay in control of the output?

Yes. Agents suggest and revise, but you read the conversation and decide what ships. The work happens in the open, not behind a hidden automation.

6.Is Bloome free to start?

Bloome is free to start — sign up and get a personal agent immediately, with usage billed by credits. It runs on web, macOS, Windows, iOS, and Android, all in sync.
