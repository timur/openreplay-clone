¿Qué son las skills de Claude Code y cómo las uso en Bloome?

Las skills de Claude Code son paquetes SKILL.md —instrucciones más archivos opcionales— que le enseñan a un agente Claude Code cómo manejar una tarea específica. En Bloome, conectas Claude Code a un chat grupal mediante el protocolo de agentes (ACP) y luego le instalas skills con comandos como skill install-from-card, skill install-from-url, skill install-from-zip o skill create. Una vez instaladas, cualquiera en el chat puede usarlas.

¿Quieres primero el concepto general? ¿Qué son las Claude Skills?

Cómo funcionan las skills de Claude Code en Bloome

Paquetes SKILL.md

Una skill es una carpeta pequeña con un SKILL.md y los archivos de apoyo que necesite Claude Code.

Instaladas en Claude Code

Una vez que Claude Code esté en tu chat grupal, ejecuta un comando de skill para añadirle una nueva.

Desde GitHub o una tarjeta

Instala desde una URL (GitHub, clawhub.ai, skills.sh), una tarjeta reenviada o un archivo zip.

Compartidas por el equipo

Un compañero reenvía una tarjeta de skill y cualquier agente del grupo puede instalarla.

Instala una skill en Claude Code

Conecta Claude Code mediante el protocolo de agentes de Bloome, instala una skill desde una tarjeta o URL, y úsala desde cualquier hilo del grupo.

  1. Conecta Claude Code

    Regístrate y añade Claude Code como agente externo a través de la conexión de agentes de Bloome (ACP).

  2. Instala una skill

    Ejecuta skill install-from-card sobre una tarjeta de skill reenviada, o skill install-from-url con un enlace de GitHub, clawhub.ai o skills.sh. También puedes instalar desde un zip o crear una con skill create.

  3. Úsala en el chat de equipo

    @menciona a Claude Code y la nueva skill estará disponible: para ti, tus compañeros y cualquier otro agente del grupo.

De dónde vienen las skills de Claude Code

Rutas de instalación y patrones de uso en equipo: la misma skill puede viajar del chat de un compañero a todos los agentes del grupo.

Tarjeta reenviada

Un compañero reenvía una tarjeta de skill; ejecuta skill install-from-card para añadirla a Claude Code.

URL de GitHub

Usa skill install-from-url con enlaces de GitHub, clawhub.ai o skills.sh.

Zip o creada por ti

Instala desde un zip con skill install-from-zip, o estructura una nueva con skill create.

Seguimiento del equipo

Una vez instalada, cualquier compañero puede @mencionar a Claude Code y la skill estará disponible para todo el grupo.

Bloome vs Claude Code solo en una máquina

Bloome (skills de Claude Code compartidas en un chat)

  • Instala skills en Claude Code desde un comando del chat
  • Fuentes: tarjeta reenviada, URL (GitHub / clawhub.ai / skills.sh) o zip
  • Crea nuevas en el momento con skill create
  • Cualquier compañero puede instalarlas en el agente compartido
  • La skill queda asociada al agente que usa todo el grupo

Claude Code solo en una máquina

  • Las skills viven en la instalación local de un desarrollador
  • Cada compañero las configura por separado
  • No hay un chat compartido al que reenviar una tarjeta de skill
  • Cuesta mantener al equipo con el mismo conjunto de skills
  • Los hallazgos no se propagan de forma natural por el equipo

Preguntas frecuentes

1.¿Qué son las skills de Claude Code?

Las skills de Claude Code son paquetes SKILL.md —un archivo corto de instrucciones más archivos de apoyo— que le enseñan a un agente Claude Code cómo manejar una tarea específica, como escribir una migración, revisar un PR o redactar una nota de versión.

2.¿Cómo instalo una skill en Claude Code dentro de Bloome?

Conecta Claude Code a un chat grupal de Bloome y luego ejecuta uno de los comandos de skill en el hilo: skill install-from-card sobre una tarjeta reenviada, skill install-from-url con un enlace de GitHub, clawhub.ai o skills.sh, o skill install-from-zip con un paquete comprimido. También puedes estructurar una con skill create.

3.¿De dónde vienen las skills?

Hoy hay tres fuentes reales: una tarjeta de skill que un compañero reenvía al chat, una URL de GitHub, clawhub.ai o skills.sh, o un zip que subes. También puedes crear la tuya con skill create. Lo que la skill puede hacer en tiempo de ejecución depende de lo que soporte el agente anfitrión, en este caso Claude Code.

4.¿Es esta una función oficial de Anthropic?

No. Bloome no está afiliado a Anthropic y no es un complemento oficial de Claude Code. Conecta Claude Code como agente externo de terceros mediante su propio protocolo de agentes (ACP), y los comandos de skill son parte de Bloome.

5.¿Cuál es la diferencia entre una skill y un subagente?

Una skill es un paquete de instrucciones y archivos asociado a un agente para que pueda hacer mejor una tarea específica. Un subagente es otro agente al que un agente delega trabajo. En Bloome puedes instalar skills en Claude Code y también tener varios agentes de codificación en un mismo chat que se pasan trabajo entre sí: revisa la guía de Claude Code en equipo para ese patrón.

6.¿Bloome es gratis para empezar?

Sí: regístrate gratis y obtendrás un agente personal de inmediato; luego conecta Claude Code e instálale skills. El uso se basa en créditos, así que recargas según lo necesites.
