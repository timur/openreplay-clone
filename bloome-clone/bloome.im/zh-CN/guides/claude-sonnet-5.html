Claude Sonnet 5 是什么？

Anthropic 的 Claude Sonnet 5 是一个更强的中等规模模型，专为低成本运行 agent 打造 —— 它会做规划、会用浏览器和终端工具，并以极低的成本自主完成任务。

Claude Sonnet 5 是什么？

Claude Sonnet 5 是 Anthropic 于 2026 年 6 月 30 日发布的中等规模模型。它专为低成本运行 agent 打造：能规划多步骤工作，能使用浏览器、终端这类工具，并自主执行任务 —— 而这类 agentic 工作，几个月前还需要更大、更贵的模型才能胜任。Anthropic 给出的定价约为每百万输入 token 2 美元、每百万输出 token 10 美元（限时优惠价，截至 2026 年 8 月 31 日），并把它设为 Free 和 Pro 用户的默认模型。核心卖点：以接近前沿的能力运行 agent，成本却低得多。

Sonnet 5 有哪些看点

能力强的中等规模模型

Anthropic 称它在许多任务上已接近自家顶级模型的水平，同时仍处在更小、更便宜的档位。

为 agent 工作而生

它会规划多步骤任务，并调用工具 —— 浏览器、终端 —— 去动手而不只是回答，因此能自主跑起 agentic 任务。

运行成本低

每百万输入 / 输出 token 约 2 / 10 美元（限时优惠价，截至 2026 年 8 月 31 日）—— 过去需要更贵模型才能做的 agent 工作。

Free 与 Pro 的默认模型

Anthropic 把 Sonnet 5 设为 Free 和 Pro 用户的默认模型，因此大多数人开箱即用。

更便宜的能力型模型，改变了难点所在

像 Sonnet 5 这样的发布，有意思的地方不只是降价 —— 而是降价所解锁的东西。当一个能力强、能做 agentic 工作的模型变便宜，运行一个 AI agent 就不再是奢侈品，同时跑好几个也变得现实。这就把真正的难题挪了位置。它不再是「我付得起一个够聪明的模型吗？」—— 而是「我怎么让一队 agent 真正协作把事办成？」。一个 agent 再便宜、再强，交给你的也只是单一产出。Bloome 面向的正是模型之上的那一层：它以 agent 为原生，且支持多模型，agent 是聊天里的一等成员，你按 credit 为实际使用付费 —— 于是一整队各有所长的 agent 可以低成本地为同一个目标协作。当底层模型越来越便宜、越来越强，这个协作层才是杠杆所在。

常见问题

1.Claude Sonnet 5 是什么？

它是 Anthropic 于 2026 年 6 月 30 日发布的中等规模模型，专为低成本运行 agent 打造 —— 会规划、会用浏览器和终端工具，并自主工作。Anthropic 给出的定价约为每百万输入 / 输出 token 2 / 10 美元（限时优惠价，截至 2026 年 8 月 31 日），并把它设为 Free 和 Pro 用户的默认模型。

2.Claude Sonnet 5 多少钱？

Anthropic 的限时优惠价约为每百万输入 token 2 美元、每百万输出 token 10 美元，截至 2026 年 8 月 31 日。当前权威定价请以 Anthropic 官网为准。

3.Sonnet 5 和 Opus 有什么区别？

Opus 是 Anthropic 最大、能力最强的档位；Sonnet 5 是中等规模模型，定位是在许多任务上以低得多的价格逼近那种能力。实际中，当你想要强劲的 agentic 表现又不想付顶级档位的成本时，就会选 Sonnet 5。

4.更便宜的 agent 模型对「一队 agent」意味着什么？

它让同时运行多个 agent 变得现实，于是瓶颈从模型成本转向协调 —— 如何让几个 agent 分工并协作。而这个协作层正是 Bloome 提供的：一个以 agent 为原生、支持多模型、按 credit 计费的地方，让一队 agent 在同一个聊天里协同工作。
