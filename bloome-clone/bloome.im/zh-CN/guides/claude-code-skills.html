Claude Code skill 是什么？怎么在 Bloome 里使用？

Claude Code skill 就是一组 SKILL.md 包 — 一份说明加上可选的辅助文件 — 用来教 Claude Code 处理某类具体任务。在 Bloome，你先通过 agent 协议（ACP）把 Claude Code 接入群聊，然后用 skill install-from-card、skill install-from-url、skill install-from-zip 或 skill create 给它安装 skill。装好之后，群里的所有人都能用。

Claude Code skill 在 Bloome 里的工作方式

SKILL.md 包

一个 skill 就是一个小目录，里面包含 SKILL.md 和 Claude Code 用得上的辅助文件。

安装到 Claude Code 上

Claude Code 一旦进入你的群聊，就可以在群里跑 skill 命令给它装上新的能力。

来自 GitHub 或卡片

支持从 URL（GitHub、clawhub.ai、skills.sh）、转发的 skill 卡片或 zip 文件安装。

团队共享

队友转发一张 skill 卡片，群里任何 agent 都能装上这同一个 skill。

给 Claude Code 装一个 skill

通过 Bloome 的 agent 协议接入 Claude Code，从卡片或 URL 安装 skill，再在群里任意 thread 中调用。

  1. 接入 Claude Code

    注册后，通过 Bloome 的 agent 连接（ACP）把 Claude Code 作为外部 agent 添加进来。

  2. 安装 skill

    在转发的 skill 卡片上跑 skill install-from-card；或者用 skill install-from-url 配合 GitHub、clawhub.ai、skills.sh 链接。也可以从 zip 安装，或者用 skill create 自己起草一个。

  3. 在团队聊天里使用

    @提及 Claude Code，新装的 skill 就能用 — 你、队友以及群里其他 agent 都能用。

Claude Code skill 的来源

覆盖安装路径和团队协作模式 — 同一个 skill 能从一位队友的聊天传到群里所有 agent 上。

转发卡片

队友转发了一张 skill 卡片，跑 skill install-from-card 就能装到 Claude Code 上。

GitHub 链接

用 skill install-from-url 配合来自 GitHub、clawhub.ai 或 skills.sh 的链接。

zip 或自建

用 skill install-from-zip 从 zip 安装，或用 skill create 起草一个新的。

团队跟进

装上之后，任何队友 @提及 Claude Code 都能用上这个 skill — 整个群共享。

Bloome 与单机版 Claude Code 的差别

Bloome（在聊天里共享 Claude Code skill）

  • 在聊天里发命令就能给 Claude Code 装 skill
  • 来源：转发卡片、URL（GitHub / clawhub.ai / skills.sh）或 zip
  • 用 skill create 现场起草新 skill
  • 任何队友都能给共享的 agent 装 skill
  • skill 一直挂在整个团队都用的那个 agent 上

单机版 Claude Code

  • skill 只存在于某个开发者本地的安装中
  • 每位队友各自单独配置
  • 没有可以转发 skill 卡片的共享聊天
  • 团队不容易保持同一套 skill 集
  • 新发现的好 skill 难以自然扩散到全团队

常见问题

1.Claude Code skill 是什么？

Claude Code skill 就是 SKILL.md 包 — 一份简短的说明文件加上若干辅助文件 — 用来教 Claude Code agent 处理某类具体任务，比如写一次数据库迁移、审 PR、起草 release note。

2.在 Bloome 里怎么给 Claude Code 装 skill？

把 Claude Code 接入 Bloome 群聊，然后在 thread 里跑对应命令：在转发的 skill 卡片上用 skill install-from-card；用 skill install-from-url 配合 GitHub、clawhub.ai 或 skills.sh 链接；或者用 skill install-from-zip 装本地 zip 包。也可以用 skill create 起草一个新的。

3.skill 都从哪里来？

目前有三种真实来源：队友转发到聊天里的 skill 卡片；来自 GitHub、clawhub.ai 或 skills.sh 的 URL；以及你上传的 zip 包。你也可以用 skill create 自己写一个。skill 在运行时能做什么，取决于宿主 agent — 这里就是 Claude Code — 实际支持什么。

4.这是 Anthropic 的官方功能吗？

不是。Bloome 与 Anthropic 没有从属关系，也不是 Claude Code 的官方插件。它通过自己的 agent 协议（ACP）把 Claude Code 作为第三方外部 agent 接入，skill 命令是 Bloome 自身的能力。

5.skill 和 subagent 有什么区别？

skill 是挂在某个 agent 上的一组说明和文件，让它把某件事做得更专；subagent 则是另一个 agent，由其他 agent 把任务委派过来。在 Bloome 里你既可以给 Claude Code 装 skill，也可以在同一个聊天里跑多个编码 agent 互相分工 — 后者参见 Claude Code 团队指南。

6.Bloome 可以免费开始吗？

可以 — 免费注册即可立刻获得个人 agent，然后接入 Claude Code 并给它装 skill。使用按 credit 计费，可以按需充值。
