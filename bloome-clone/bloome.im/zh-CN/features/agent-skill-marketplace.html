Agent Skill 市场

一座开放的 agent skill 能力库。转发一张 skill 卡片、贴一个 URL，或上传一个 zip — 你拥有的任意 agent 就能学会新的能力。

Bloome 的 agent skill 市场是什么？

Bloome 的 agent skill 市场是一座开放的 skill 能力库 — 这里的 skill 都是 SKILL.md 包，给 agent 注入某种专门能力。skill 在 Bloome 里是一等对象：你可以从聊天里的 skill 卡片、公开 URL（GitHub、clawhub.ai、skills.sh）或上传的 zip 把它装到你拥有的任意 agent 上，也可以从零自己写。

想找的是 agent 库而不是 skill 库？ 查看 AI agent 市场

skill 库能让你做什么

浏览与搜索

发现别人发布的 SKILL.md 包，搜索你需要的能力。

装到任意 agent 上

把 skill 加到你拥有的任意 agent 上 — 个人 agent 或从 Explore 克隆来的 agent 都行。

通过 skill 卡片分享

在聊天里转发一张 skill 卡片，对方一键就能装到自己的 agent 上。

自带 skill

从 GitHub / clawhub.ai / skills.sh 的 URL 安装、上传 zip，或者直接在聊天里写 SKILL.md。

运作方式

  1. 找到或分享 skill

    在 skill 库里挑一个，贴一个公开 URL，或者把别人发到聊天里的 skill 卡片转发出去。

  2. 装到你的 agent 上

    点击安装，选择把这项能力交给哪个 agent — SKILL.md 包就会被加到那个 agent 上。

  3. 在聊天里使用

    在任意聊天里 @提及 这个 agent，新 skill 就能与它原有的工具和提示一起工作。

走进 skill 库

把 skill 当作可复用的积木 — 发现、共享、安装、定制。

一份能力目录

skill 按用途归类 — 研究、写作、代码、数据、规划 — 你扫一眼网格就能挑。

聊天里的 skill 卡片

skill 卡片是一种系统消息，可以转发给队友 — 一键就能装到对方的 agent 上。

随处可装

支持来自 GitHub、clawhub.ai 或 skills.sh 的公开 URL，你上传的 zip，或者从零写的 SKILL.md。

在团队中叠加 skill

给不同的 agent 装不同的 skill，再让它们在同一个工作 thread 里协同。

开放的 skill 库 vs 单机配置

Bloome（开放库 + 聊天内分享）

  • 在应用内浏览公开的 skill 库
  • skill 是一等对象 — 装到你拥有的任意 agent
  • 聊天里转发 skill 卡片，对方一键安装
  • 支持 GitHub / clawhub.ai / skills.sh 的 URL 安装
  • 支持上传 zip，或现场编写 SKILL.md

封闭式 / 单机 skill 配置

  • 没有共享库 — 你得自己在各种仓库里翻
  • skill 绑死在一台机器、一个工具上
  • 没有把 skill 在聊天里发给队友的方式
  • 每次安装都得手动操作文件系统
  • 同一个 skill 难以在多个 agent 间复用

常见问题

1.agent skill 是什么？

skill 是一个 SKILL.md 包，给 agent 注入某种专门能力 — 比如一种研究流程、一种写作模式或某个领域的处理流程。在 Bloome，skill 是一等对象，可以装到你拥有的任意 agent 上。

2.怎么把 skill 装到我的 agent 上？

在 skill 库里打开一个 skill，或者在聊天里打开一张 skill 卡片，点击安装，再选择把它给哪个 agent。SKILL.md 包就会加到那个 agent 上，下次 @提及 它时就能用。

3.可以把 skill 分享给团队吗？

可以。skill 卡片在 Bloome 聊天里是一种系统消息 — 把它转发给好友或群组，对方一键就能装到自己的 agent 上。

4.skill 都从哪里来？

四种来源：聊天里的 skill 卡片；来自 GitHub、clawhub.ai 或 skills.sh 等的公开 URL；你上传的 zip 文件；以及你在 Bloome 内从零编写的 SKILL.md。

5.和 agent 市场有什么区别？

agent 市场（Explore）是一个可克隆的 agent 库；skill 市场则是一个能力库 — 你把 skill 装到已经拥有的 agent 上，让它学会新能力，而不是再克隆一个新 agent。

6.skill 市场要付费吗？

Bloome 可以免费开始 — 浏览 skill 库和安装 skill 都属于产品本身的能力。使用按 credit 计费，需要时充值即可。
