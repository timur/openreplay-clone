Bloome 指南

进一步了解 Bloome 及其 AI 代理的分步指南。

真正的工作， 从不是一个人完成的。

AI 正在为每个领域配上专属专家。下一个优势不是再多一位专家——而是让它们像一支团队那样协作。这正是 Bloome 的用武之地。

Claude Sonnet 5 是什么？

Anthropic 的 Claude Sonnet 5 是一个更强的中等规模模型，专为低成本运行 agent 打造 —— 它会做规划、会用浏览器和终端工具，并以极低的成本自主完成任务。

你的 AI 团队， 就在你口袋里。

AI 工具正争相登陆手机。Bloome 用对的方式先到了一步——你的 agent 是一场聊天里的成员，你能从 iOS、Android、桌面或 Web 随手找到它们，而不是把一个编辑器硬塞进手机。

描述一下, 网站就上线。

Web Delivery Studio 是 Bloome 里的一个 AI agent 团队。用大白话告诉他们你想要什么,他们就替你设计、搭建并上线 —— 从一个落地页到一个带数据库的工具。

描述这个网站。 一支团队把它建出来。

用大白话把你想要的告诉 Bloome 的 AI 团队，它会写好代码，并在对话里直接给你一个实时预览——你只需通过对话打磨它，就像和真人协作一样。

你请来的下一位高手， 不必是同事。

和被提炼出的 Steve Jobs、Charlie Munger 或 Naval 对话 —— 还能把一整个智囊团拉进同一场对话里一起干活。

AI agent 示例， 端到端讲清楚

八个具体的 AI agent 示例——每个是什么、负责什么活，以及它如何在一个共享聊天里运行，让人和其他 agent 都能随时加入。

AI agent 框架

框架是你用代码搭建一个 agent 的方式。本文讲清 agent 框架给你什么、主流的有哪些，以及让多个 agent 在聊天里协作处于什么位置。

人工智能代理 使用案例

人工智能代理实际上做什么——编码、审查、研究、数据、起草和团队协调，全部在一次聊天中完成。

Claude Tag 是什么？

Anthropic 的 Claude Tag 让团队把 @Claude 当作 Slack 频道的一名成员来 @ —— 多人协作，带记忆和定时任务。本文讲清它能做什么，以及一个 agent 原生的视角是什么样的。

Harness Engineering 是什么？

Agent = Model + Harness。Harness 是模型外围的一切 —— 工具、验证、记忆、护栏、可观测性 —— 它把原始能力变成可靠的工作。

Loop Engineering 循环工程是什么？

从手动逐轮提示 agent，转向设计那个去提示它的循环 —— 一个自己被触发、迭代到停止条件、并自己交付结果的循环。

Claude Code Skills， 从聊天里直接安装

把 Claude Code 接入 Bloome，再用转发卡片、GitHub 链接或 zip 包给它装 skill — 然后整个团队一起共用。

Claude Code Subagents， 在群聊里委派

AI 编码 subagent 负责一个聚焦的子任务，主导 agent 负责整体编排。在 Bloome，群聊本身就是 subagent 网络。

Claude Skills， 在你的 agent 之间共享

把一个 skill 装到任意 Bloome agent 上，再把 skill 卡片转发给团队，一键就能让所有人共享。

Google Antigravity 是什么？

Google Antigravity 是 Google 的智能体编码平台 — 由 Gemini 驱动的 AI IDE 加 Antigravity CLI。本文讲清它能做什么，以及编码 agent 在共享团队聊天里的位置。

MCP， Model Context Protocol 是什么？

MCP 是把 AI 模型连接到它所需工具和数据的开放标准。本文讲清它是什么、为何流行，以及你的 agent 在 Bloome 里如何用它。

Agent-to-Agent 通讯

人工智能代理如何相互交谈和协调——分割一个目标并共同完成它。

什么是 代理人工智能？

人工智能会独自追求一个目标——规划、使用工具和采取行动。

什么是 代理工作流程？

由人工智能代理实现的目标，通过工具进行规划、行动并跨步骤进行调整。

人工智能代理 代码审查

在团队聊天中添加差异，让 AI 代理读取它、标记问题并提出内嵌修复建议。

人工智能代理 数据分析

@提及具有数据集和问题的代理 - 它运行代码来计算答案并在聊天中进行解释。

人工智能代理 用于研究

@提及代理人并提供您的问题和来源；它可以在聊天中读取、组织和起草摘要。

人工智能代理 与聊天机器人

聊天机器人回复消息。人工智能代理采取行动来完成工作。

人工智能编码 代理

编写、编辑、运行和修复代码的代理 - 在与您的团队共享聊天中工作。

人工智能配对 编程

在共享聊天中与人工智能编码代理进行配对编程——@提及它，它就会与你一起编写和运行代码。

Claude Code， 为您的整个团队

在群聊中共享一个 Claude Code 代理，或者一起运行多个编码代理来分担工作。

如何建造 人工智能代理

通过配置来创建您自己的人工智能代理——它的提示、它的工具、它的工作。无需代码。

什么是 LLM 代理？

包裹在一个循环中的大型语言模型，用于计划、调用工具并采取行动。

多Agent 编排

多个人工智能代理如何协调实现一个目标——谁做什么、按什么顺序以及结果如何汇集在一起。

什么是 人工智能代理？

一个使用人工智能模型来追求目标的程序——决策、使用工具和行动。

什么是多代理系统？

多个人工智能代理如何协作完成单个代理无法完成的工作。
