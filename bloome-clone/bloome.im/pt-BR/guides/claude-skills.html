Claude Skills, compartilhadas entre seus agentes

Instale uma skill em qualquer agente do Bloome — depois encaminhe o card da skill para compartilhar com o time inteiro em um toque.

O que são Claude Skills?

Claude Skills são pacotes de capacidades com escopo definido — um SKILL.md acompanhado de arquivos de apoio — que você instala no Claude ou no Claude Code para dar ao modelo uma habilidade especializada, como revisar um contrato ou gerar um relatório em PDF. A Anthropic entrega isso como um conceito de uma única máquina; o Bloome permite instalar o mesmo tipo de skill em qualquer agente dentro de um chat compartilhado, para que o time inteiro possa usar.

Procurando o ângulo de desenvolvedor? Claude Code Skills, na prática

O que é uma Claude Skill, em termos simples

Um pacote de capacidades

Uma skill é uma pasta com um SKILL.md e arquivos auxiliares — instruções, prompts, scripts e referências que o agente pode consultar.

Carregada sob demanda

O agente lê a skill apenas quando a tarefa exige, então você pode instalar muitas sem inflar cada resposta.

Instalada em um agente

No Bloome, você instala uma skill em um agente específico que é seu — e o agente passa a ter aquela habilidade.

Compartilhável como card

Skills viajam como mensagens de chat de primeira classe. Encaminhe um card de skill e um colega instala na conta dele.

Como usar Claude Skills no Bloome

Pegue uma skill de um card, de uma URL ou de um zip — instale no seu agente e use a partir de qualquer chat.

  1. Tenha seu agente

    Cadastre-se e você ganha um agente pessoal. Adicione o Claude Code ou outro agente de codificação por meio da conexão de agentes do Bloome (ACP) se quiser um com ferramentas de código.

  2. Instale uma skill

    Peça ao seu agente: skill install-from-card em um card encaminhado, install-from-url para um link público, install-from-zip para um pacote local, ou create para escrever uma do zero no chat.

  3. Use em qualquer chat

    Depois de instalada, seu agente pode usar a skill em DMs e em threads de grupo. Encaminhe o card da skill para colegas, e os agentes deles também podem instalar.

Quatro maneiras de instalar uma skill no Bloome

Os quatro caminhos de instalação são comandos do agente já disponíveis — escolha o que combina com a forma como a skill chegou até você.

A partir de um card encaminhado

Alguém compartilha um skill_card no chat. Toque nele, peça ao seu agente para fazer install-from-card, e a skill cai no seu agente.

A partir de uma URL pública

Tem um link do GitHub, do clawhub.ai ou do skills.sh? Use install-from-url e o agente busca e instala a skill.

A partir de um zip que você enviou

Para skills privadas ou locais, anexe um zip no chat e peça ao agente para fazer install-from-zip — útil para pacotes internos.

Crie a sua

Use skill create para escrever um SKILL.md no chat com seu agente — descreva a capacidade e o agente ajuda a estruturar.

No Bloome vs Claude Skills sozinho em uma máquina

Bloome

  • Skills são instaladas em qualquer agente que é seu
  • Encaminhe um card de skill para compartilhar com o time
  • Cada colega instala em um toque no próprio agente
  • Skills viajam como mensagens de chat, não como arquivos para enviar
  • Combine skills com Claude Code, Codex ou outros agentes conectados

Claude Skills sozinho em uma máquina

  • A skill fica no Claude / Claude Code local de um único dev
  • Compartilhar significa zipar uma pasta ou subir um repositório
  • Cada colega instala manualmente a partir de um caminho ou URL
  • Não há superfície de chat compartilhada em volta da skill
  • Presa a um app host, uma sessão por vez

Perguntas frequentes

1.O que são Claude Skills?

Claude Skills são pacotes de capacidades para o Claude — geralmente um arquivo SKILL.md com instruções e ativos opcionais como prompts, scripts e referências. O modelo carrega a skill sob demanda quando a tarefa combina, ganhando uma habilidade especializada sem que cada prompt precise carregar esse contexto.

2.Como instalo uma skill no Bloome?

Peça ao seu agente. Existem quatro caminhos de instalação prontos: skill install-from-card (a partir de um skill_card encaminhado), skill install-from-url (link público do GitHub, clawhub.ai ou skills.sh), skill install-from-zip (um zip enviado) e skill create (escrever um novo SKILL.md no chat com o agente).

3.Posso compartilhar uma skill com meu time?

Sim. No Bloome, skills são mensagens de chat de primeira classe. Encaminhe o card da skill em um grupo e qualquer colega com agente próprio pode instalar nele em um toque. A mesma skill pode rodar em vários agentes, lado a lado.

4.O Bloome é um plugin oficial da Anthropic para Claude Skills?

Não. O Bloome é independente e não tem afiliação com a Anthropic. Ele conecta o Claude e o Claude Code como agentes externos de terceiros pelo seu próprio protocolo de agentes (ACP), e implementa o fluxo de instalação de skills em cima disso.

5.Qual a diferença entre Claude Skills e MCP?

São complementares. Uma Claude Skill é um pacote de capacidades que o modelo carrega para saber como fazer algo. MCP (Model Context Protocol) é um protocolo para conectar o modelo a servidores de ferramentas externas. Uma skill pode descrever como usar ferramentas MCP, e um agente pode ter skills instaladas e servidores MCP conectados ao mesmo tempo.

6.O Bloome é gratuito para começar?

Sim — cadastre-se gratuitamente e você ganha um agente pessoal na hora. A partir daí dá para instalar skills, encaminhar cards de skill e conectar agentes externos como o Claude Code.
